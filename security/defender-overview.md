# Microsoft Defender in a Modern Workplace

## What this is
This document provides a practical overview of Microsoft Defender products and how they support security across Microsoft 365 environments.

Microsoft Defender is not a single product, but a family of security services that protect users, devices, email, and cloud apps.

---

## Why Defender matters
Modern attacks target:
- User identities
- Endpoints
- Email and collaboration tools
- Cloud applications

Defender provides:
- Threat detection
- Automated response
- Centralised visibility
- Protection across Microsoft 365

---

## Core Defender components

### Microsoft Defender for Endpoint
Protects Windows devices by:
- Detecting malware and suspicious behaviour
- Providing endpoint detection and response (EDR)
- Supporting device isolation and investigation

---

### Microsoft Defender for Office 365
Protects email and collaboration by:
- Blocking phishing and malware
- Scanning links and attachments
- Investigating user-reported threats

---

### Microsoft Defender for Cloud Apps
Provides:
- Visibility into cloud app usage
- Session controls and monitoring
- Data protection for cloud services

---

### Microsoft Defender XDR
Brings together:
- Identity
- Endpoint
- Email
- Cloud app signals  
into a single investigation experience.

---

## How Defender fits into Microsoft 365

Defender works with:
- Entra ID (identity signals)
- Intune (device compliance)
- Conditional Access (access enforcement)

Together they support Zero Trust:
- Verify identity
- Verify device
- Detect and respond to threats

---

## What to validate

A Microsoft 365 administrator should regularly check:
- Defender alerts and trends
- Endpoint protection coverage
- Email protection effectiveness
- Cloud app discovery
- Response and remediation activity

---

## How this supports a secure Modern Workplace

Defender allows:
- Threats to be detected quickly
- Incidents to be investigated centrally
- Devices and users to be protected automatically
- Security teams to work more efficiently

It turns Microsoft 365 into a monitored, defended platform rather than just a set of apps.
